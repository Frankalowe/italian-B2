# Masterclass Vocabulary Sheet: Direct & Indirect Object Pronouns (A2 Level)

**Unit Focus:** Direct & Indirect Object Pronouns (Pronomi Diretti e Indiretti)
**Contextual Theme:** Dining, Shopping, and Social Interactions

This vocabulary sheet is designed to provide the practical language necessary to use object pronouns correctly in everyday Italian conversations. Focus on the context provided in the examples to understand *when* and *how* to use these words.

## 💡 Grammar Note for Pronoun Placement (A2 Focus)

When learning these pronouns, remember that they must be placed **before the conjugated verb** in Italian.

**Example Application:**

1.  **Direct Object:** If you are talking about the menu (*il menù* - masculine singular), you use *lo*.
    *   *Io voglio **lo**.* (I want **it**.)
2.  **Indirect Object:** If you are talking about the waiter (*il cameriere* - masculine singular), you use *gli*.
    *   *Io do il libro a lui.* $\rightarrow$ *Io **gli** do il libro.* (I give the book to **him**.)

**Practice Tip:** Always identify the noun you are replacing (the object) before deciding which pronoun (*lo, la, li, le, gli, loro*) to use!
