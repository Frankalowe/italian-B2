## Summary Grammar Note for Vocabulary Use

When using the vocabulary above with the *Passato Prossimo*, remember the core rule:

1.  **Identify the Verb:** Determine if the action involves movement, a state of being, or possession.
2.  **Choose the Auxiliary:**
    *   For verbs of **movement** (*andare, venire, partire*), **use *essere*** (e.g., *Sono andato*).
    *   For verbs of **possession/action** (*avere, mangiare, prenotare*), **use *avere*** (e.g., *Ho mangiato*).
3.  **Agreement:** If you use *essere*, the past participle **must agree** in gender and number with the subject (e.g., *Siamo andati* vs. *Sono andata*).
